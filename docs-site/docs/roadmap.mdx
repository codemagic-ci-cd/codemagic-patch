# Roadmap

This page tracks the near-term direction for Codemagic Patch. Priorities can change as we learn more, but major shifts will be reflected here.

Have a request for us? Open a discussion on GitHub: [codemagic-patch discussions](https://github.com/codemagic-ci-cd/codemagic-patch/discussions).

## In progress

- Better release diagnostics and error surfacing in dashboard and CLI
- Smoother first-run flow from app creation to first release
- Expanded docs examples for common CI pipelines

## Next up

- More granular rollout controls and clearer rollout visibility
- Team-level governance improvements (roles, safer production actions)
- Stronger migration guidance for Expo Updates and CodePush users

## Planned

- Additional delivery adapter guidance and hardening docs
- Better long-term operations guidance (monitoring, backup cadence, recovery drills)
- More reference coverage for advanced SDK integration patterns

# Codemagic Patch

Self-hosted over-the-air updates for React Native. Ship JavaScript and asset changes to installed apps without an app-store review when the change lives in your JS bundle.

## Try it first

New here? Work through the **[Self-host guide](/docs/setup/self-host)** to run the server, then [connect your React Native app](/docs/setup/native-setup) and [publish your first release](/docs/setup/first-release).

## How the docs are organized

- **Introduction**: how Patch works, core concepts
- **Setup**: self-host, CLI, SDK integration, first release
- **Using Patch**: publishing and rollouts, code signing, delivery
- **Reference**: CLI commands, configuration, server operations

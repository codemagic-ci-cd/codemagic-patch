# Changelog

## Dashboard sample data

*July 14, 2026*

Local evaluation stacks now seed **Example Data** with Staging and Production releases and metrics, so the dashboard looks populated before you publish your first update.

## Local evaluation stack

*July 8, 2026*

One-command local evaluation: `./scripts/local-eval/up.sh` brings up the API, dashboard, Postgres, and MinIO on localhost, with instant sign-in and a seeded demo app for the on-device walkthrough.

- Dashboard at `http://localhost:8080`, API at `http://localhost:3000`
- On-device demo under `examples/on-device-demo` to watch an OTA apply on simulator or emulator

## CLI command builder

*July 3, 2026*

Interactive command builder in the web dashboard when publishing a release. Pick platform, deployment, rollout, and other flags in the UI and copy a ready-to-run `cmpatch release-react` command.

- **New release → Via CLI**: form-based builder that assembles `cmpatch release-react` from your choices (platform, deployment, rollout %, target binary version, release notes, mandatory flag)
- Copy-to-clipboard on the generated command for pasting into a terminal or CI job

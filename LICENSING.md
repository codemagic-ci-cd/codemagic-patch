# Licensing

Codemagic Patch is distributed under two licenses, split by component. The
client-facing pieces (the React Native SDK and the CLI) are permissively
licensed under **Apache License 2.0**, while the server and its supporting
components are licensed under the **Codemagic Server License** (a
custom license based on [Functional Source License 1.1](https://fsl.software/)
that converts to Apache 2.0).

Each directory's own `LICENSE` file governs it; the root `LICENSE` applies
to anything without one. This file is only a human-readable summary.

## Codemagic Server License — key terms

- **Free** up to **1,000,000 Monthly Active Users** (measured as unique devices
  that request an update in a month); above that, you need a commercial license.
- **No Competing Use**: self-hosting it for your own apps is fine
  (production and commercial use included), but you can't offer it as a
  commercial product or service that substitutes for Codemagic Patch or provides
  substantially similar functionality.
- **Converts to Apache-2.0** two years after each version is published.

For a commercial license above the Monthly Active Users threshold, contact
Codemagic (Nevercode Ltd) — <https://codemagic.io>.
